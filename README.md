## 📋 Descrição

Este é um projeto de uma "landing page" com o tema "Halloween".
Esse projeto foi apresentado na terceira aula prática do módulo 02 do curso de front-end do "Vai na Web". Sua principal proposta era colocar em prática os conhecimentos em estilização com "Styled Components" e lógica de programação.
O modelo a ser seguido foi apresentado na plataforma de design "Figma".

### 🚀 Funcionalidades

As funcionalidades disponíveis para os usuários estão listadas abaixo:

- Visualizar imagens de fantasias de uma das festividades mais famosas do ano
- Acessar uma página simples sobre o Halloween

## ✏️ Aprendizado

Ao fazer esse projeto, aprendi a:

- Manipular conjuntos de elementos utilizando a função "map" com JavaScript
- Praticar o conceito de componentização com React JavaScript
- Revisar a estilização de elementos utilizando a biblioteca "Styled Components" do React
- Criar uma página simples sobre o dia das bruxas, utilizando a biblioteca React JavaScript
